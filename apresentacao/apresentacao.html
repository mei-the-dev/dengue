Introdução

Objetivos e metodologia do estudo epidemiológico

Objetivo

Identificar padrões temporais e espaciais de sincronização epidêmica entre municípios do RJ usando técnicas de Análise Topológica de Dados (TDA).

Fonte de Dados

DATASUS/SINAN - Sistema de Informação de Agravos de Notificação. Casos confirmados de dengue por semana epidemiológica.

Metodologia

Normalização de séries, cálculo de distâncias L1/L2, construção de complexos simpliciais, PCA e clusterização.

Semanas Epidemiológicas

Padrão OMS/CDC: semanas de domingo a sábado, numeradas de 1 a 52/53 por ano epidemiológico.

Normalização das Séries

Métodos para comparação justa entre municípios de diferentes tamanhos

Taxa de Incidência

  • Casos ÷ População × 100.000
  • Permite comparar municípios de diferentes tamanhos
  • Padrão epidemiológico internacional

Área Unitária

  • Normaliza pelo total de casos no período
  • Soma da série = 1 (área unitária)
  • Foca no padrão temporal, não na magnitude

Distância Mínima

Distância L1 (Manhattan)

Soma dos valores absolutos das diferenças:

dL1(x, y) = Σ |xi - yi|

Mais robusta a outliers. Penaliza igualmente desvios pequenos e grandes.

Distância L2 (Euclidiana)

Raiz da soma dos quadrados:

dL2(x, y) = √(Σ (xi - yi)²)

Penaliza mais fortemente grandes desvios. Padrão geométrico.

07

KeplerMapper

Visualização topológica interativa dos dados epidemiológicos

O que é KeplerMapper?

KeplerMapper é uma implementação Python do algoritmo Mapper para TDA. Ele cria uma rede simplificada que preserva a estrutura topológica dos dados, permitindo identificar clusters, loops e estruturas de alta dimensão.

  • Nós: Representam grupos de municípios similares
  • Arestas: Conectam grupos com municípios em comum
  • Cores: Indicam propriedades médias (distância, componente PCA)
  • Tamanho: Proporcional ao número de municípios no grupo

08

Conclusões

Principais descobertas e contribuições do estudo

Padrões Identificados

Clusters bem definidos de municípios com comportamentos epidêmicos similares, sugerindo fatores regionais comuns.

Sincronização

Pares de municípios altamente sincronizados foram identificados, potencialmente devido a mobilidade urbana ou condições climáticas.

Metodologia

TDA provou-se eficaz para análise epidemiológica, oferecendo insights não capturados por métodos tradicionais.

Aplicação

Resultados podem orientar políticas de saúde pública coordenadas entre municípios do mesmo cluster.

Trabalhos Futuros

  • Incluir dados climáticos (precipitação, temperatura)
  • Análise de persistência homológica
  • Modelos preditivos por cluster
  • Expandir para outros estados
  • Integrar dados de mobilidade urbana
